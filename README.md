# Games101 Experiment4

## 实验内容

- 本次实验较为简单，即为使用递归去实现贝塞尔曲线
- 主要思路即为，向函数传入一个控制点的数组，之后根据t求出n个控制点的n-1个新的点。之后再将这n-1个新的点传入递归函数中。当数组中只剩下一个点的时候，递归终止。
